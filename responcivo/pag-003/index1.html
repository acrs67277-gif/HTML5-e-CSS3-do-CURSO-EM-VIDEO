Media Queries (Consultas de Mídia, em português) são um recurso fundamental do CSS3 (Cascading Style Sheets) que permite que você aplique estilos CSS diferentes em seu site, dependendo das características do dispositivo que está sendo usado para visualizá-lo. Elas são a base do Web Design Responsivo, que é a prática de criar sites que se adaptam e oferecem uma experiência de visualização otimizada em uma ampla variedade de dispositivos, como:

  • Telas de computadores (desktops)
  • Tablets
  • Smartphones
  • Impressoras
  • Dispositivos de fala (leitores de tela)

Como funcionam?

Uma Media Query consiste em:

  • 1. Tipo de Mídia (Media Type): O tipo de dispositivo para o qual o estilo se aplica (ex: screen para telas, print para impressão, all para todos).
  • 2. Expressão: Uma condição que verifica características específicas do dispositivo ou da área de visualização (viewport).

Se o dispositivo atender às condições da expressão, os estilos CSS definidos dentro da Media Query serão aplicados.

Exemplos de Características (Media Features):

As características que você pode testar incluem:

  • width e height (Largura e altura da área de visualização)
  • min-width e max-width (Largura mínima e máxima) - muito usados para definir os chamados breakpoints.
  • orientation (Orientação do dispositivo, como landscape - paisagem/deitado - ou portrait - retrato/em pé).
  • resolution (Resolução da tela).

Sintaxe Básica:

A sintaxe no CSS é a seguinte:

@media tipo de mídia and (expressão) {
/* Estilos CSS a serem aplicados quando a condição for verdadeira */ }

Exemplo prático:

Para mudar a cor de fundo do corpo do documento para azul apenas quando a largura da tela for no mínimo 768 pixels (tamanho comum para tablets):

@media screen and (min-width: 768px) {
body { background-color: lightblue; } }
